# Таблицы

Все **таблицы храниться** в папке **CSV** (`./CSV`)

***Базовые/изначальные*** таблицы (*пользователи*, *продукты*, *заказы*, *активности*) располагаются в папке **Original** (`./CSV/Original`)

***Модернизированные*** таблицы (полученные при помощи SQL запросов) располагаются в папке **Modern** (`./CSV/Modern`)
<br>
